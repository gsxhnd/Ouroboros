# Ourboros 桌面应用打包指南

本指南说明如何为 macOS、Windows 和 Linux 打包 Ourboros 桌面应用。

## 前置要求

### 所有平台通用

- Rust 工具链（用于编译 Rust 代码）
- Node.js/npm（用于编译前端）
- 已构建的 Release 二进制文件和 Web 前端

### 编译步骤

```bash
# 1. 编译 Rust 后端（Release 版本）
cargo build -p ourboros --release

# 2. 编译前端
cd web
npm install
npm run build
cd ..
```

编译完成后，检查输出：

- Rust 二进制: `target/release/ourboros` (macOS/Linux) 或 `target/release/ourboros.exe` (Windows)
- 前端资源: `web/dist/index.html`

## Windows 打包

### 使用方式

#### 方式 1：生成便携式可执行文件（推荐）

```powershell
.\scripts\bundle-windows.ps1
```

或显式指定输出格式：

```powershell
.\scripts\bundle-windows.ps1 -OutputType exe
```

#### 方式 2：生成 MSI 安装程序

```powershell
.\scripts\bundle-windows.ps1 -OutputType msi
```

注意：需要安装 [WiX Toolset](https://wixtoolset.org/releases/)

### 输出结果

脚本生成以下目录结构：

```
target/bundle-output/Ourboros/
├── Ourboros.exe          # 主程序
├── run.bat               # 快捷启动脚本
└── resources/
    └── web/              # 前端 UI 资源
        ├── index.html
        ├── assets/
        └── ...
```

### 运行应用

```bash
# 直接运行
.\target\bundle-output\Ourboros\Ourboros.exe

# 或使用启动脚本
.\target\bundle-output\Ourboros\run.bat
```

### 创建快捷方式

创建一个 Windows 快捷方式：

1. 右键点击 `Ourboros.exe`
2. 选择 "发送到" > "桌面（快捷方式）"
3. （可选）更改快捷方式的图标

## Linux 打包

### 使用方式

#### 方式 1：生成 tarball（推荐用于快速测试）

```bash
./scripts/bundle-linux.sh tarball
```

#### 方式 2：生成 AppImage

```bash
./scripts/bundle-linux.sh appimage
```

需要安装 [AppImageKit](https://github.com/AppImage/AppImageKit)

#### 方式 3：生成 DEB 包

```bash
./scripts/bundle-linux.sh deb
```

需要安装 `dpkg`（通常已预装）

### 输出结果

#### Tarball 输出

```
target/bundle-output/
├── Ourboros-0.1.0-linux-x86_64.tar.gz
└── Ourboros/
    ├── Ourboros          # 启动脚本
    ├── bin/
    │   └── ourboros      # 主程序
    ├── resources/
    │   └── web/          # 前端 UI 资源
    └── share/
        └── applications/
            └── dev.newma.ourboros.desktop
```

#### AppImage 输出

```
target/bundle-output/
├── Ourboros-0.1.0-x86_64.AppImage  # 单一可执行文件
└── Ourboros/                        # 源目录
```

#### DEB 输出

```
target/bundle-output/
├── ourboros_0.1.0_amd64.deb         # 可安装包
└── Ourboros/                        # 源目录
```

### 运行应用

**使用 Tarball：**

```bash
# 提取
tar -xzf target/bundle-output/Ourboros-0.1.0-linux-x86_64.tar.gz

# 运行
./Ourboros/Ourboros

# 或安装到系统
sudo cp -r Ourboros /opt/
/opt/Ourboros/Ourboros
```

**使用 AppImage：**

```bash
chmod +x target/bundle-output/Ourboros-0.1.0-x86_64.AppImage
./target/bundle-output/Ourboros-0.1.0-x86_64.AppImage
```

**使用 DEB：**

```bash
sudo dpkg -i target/bundle-output/ourboros_0.1.0_amd64.deb
ourboros
```

### 集成到应用菜单

```bash
# 方式 1：复制 desktop 文件到系统
sudo cp target/bundle-output/Ourboros/share/applications/dev.newma.ourboros.desktop \
  /usr/share/applications/

# 方式 2：创建系统命令链接
sudo ln -s /opt/Ourboros/Ourboros /usr/local/bin/ourboros
```

## 通用信息

### 环境变量

运行打包后的应用时，会自动设置以下环境变量：

- `OURBOROS_APP_BUNDLE=1` - 表示应用以 bundle 形式运行

### Web UI

应用启动后，Web UI 可通过以下地址访问：

- `http://127.0.0.1:8080`

### 系统托盘

如果启用了系统托盘功能（默认启用），应用会在系统托盘中显示图标。

## 故障排除

### Windows

**问题：** 提示缺少 `.exe` 文件

**解决：**

```bash
cargo build -p ourboros --release
```

**问题：** PowerShell 执行策略阻止脚本运行

**解决：**

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Linux

**问题：** 提示缺少 `web/dist/index.html`

**解决：**

```bash
cd web && npm install && npm run build && cd ..
```

**问题：** 权限被拒绝

**解决：**

```bash
chmod +x ./scripts/bundle-linux.sh
```

**问题：** `appimagetool` 或 `dpkg-deb` 未找到

**解决：**

- AppImage: 从 <https://github.com/AppImage/AppImageKit> 下载
- DEB: `sudo apt install dpkg` (Debian/Ubuntu) 或 `sudo dnf install dpkg` (Fedora)

## 版本管理

```toml
[workspace.package]
version = "0.1.0"  # 修改此处
```

## CI/CD 集成

### GitHub Actions 示例

```yaml
name: Build Desktop App

on: [push, pull_request]

jobs:
  build:
    runs-on: ${{ matrix.os }}
    strategy:
      matrix:
        os: [ubuntu-latest, windows-latest, macos-latest]

    steps:
      - uses: actions/checkout@v4
      
      - uses: actions-rust-lang/setup-rust-action@v1
      
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      
      - name: Build Rust
        run: cargo build -p ourboros --release
      
      - name: Build Web
        run: cd web && npm install && npm run build && cd ..
      
      - name: Bundle (Linux)
        if: runner.os == 'Linux'
        run: ./scripts/bundle-linux.sh tarball
      
      - name: Bundle (Windows)
        if: runner.os == 'Windows'
        run: .\scripts\bundle-windows.ps1
      
      - name: Bundle (macOS)
        if: runner.os == 'macOS'
        run: ./scripts/bundle-macos.sh
```

## 发布打包

### 创建发布版本

```bash
# 1. 更新版本号
cargo set-version 0.2.0

# 2. 为所有平台编译 Release
cargo build -p ourboros --release
cd web && npm run build && cd ..

# 3. 为所有平台生成包
./scripts/bundle-macos.sh
./scripts/bundle-linux.sh tarball
.\scripts\bundle-windows.ps1

# 4. 输出位置
# - macOS: target/Ourboros.app
# - Linux: target/bundle-output/Ourboros-0.2.0-linux-x86_64.tar.gz
# - Windows: target/bundle-output/Ourboros/
```
