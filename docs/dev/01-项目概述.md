# 01 - 项目概述

## 简介

Ourboros 是一个跨平台资源管理库工具，参考 [Eagle](https://cn.eagle.cool/) 的设计理念，专注于图片、文档、字体和音频资源的收集、组织、搜索和浏览。

## 核心目标

- 本地优先的资源管理，不转移文件，路径一一映射
- 桌面端通过 `ourboros app` 系统托盘运行，内嵌 HTTP Server，用户通过浏览器访问界面
- 也可通过 `ourboros server` 独立部署（NAS / 服务器场景）
- 文件系统实时监听，用户在文件夹的操作即时同步到 UI
- 支持多种资源类型的统一管理
- 支持多数据库后端（SQLite / PostgreSQL / MySQL，通过配置切换）

## 支持资源格式

| 类型 | 格式 |
|------|------|
| 图片 | JPEG, PNG, WebP, TIFF, BMP, GIF, SVG, HEIC, RAW (CR2/NEF/ARW/DNG) |
| 文档 | PDF |
| 字体 | TTF, OTF, WOFF, WOFF2 |
| 音频 | MP3, FLAC, WAV, AAC, OGG, M4A |

## 核心功能

- 资源索引与元数据解析（EXIF、ID3、字体信息）
- 缩略图自动生成
- 虚拟文件夹 + 智能文件夹
- 标签系统（分组、批量、自动）
- 多维度搜索（关键词、颜色、类型、尺寸、日期、评分）
- 文件系统实时监听与同步
- 多语言 / 主题切换
- 桌面端多实例、多资源库切换
- 跨平台支持（Windows / macOS / Linux）
