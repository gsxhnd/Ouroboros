# 08 - 技术依赖

## Rust Crates

| Crate | 用途 |
|-------|------|
| `axum` | HTTP 框架 |
| `tokio` | 异步运行时 |
| `tower-http` | CORS、日志、Trace 等 HTTP 中间件 |
| `sea-orm` | ORM 框架（支持 SQLite/PostgreSQL/MySQL） |
| `sea-orm-migration` | 数据库迁移 |
| `tray-icon` | 系统托盘图标 |
| `muda` | 系统托盘菜单 |
| `tao` | 跨平台事件循环 |
| `open` | 打开系统默认浏览器 |
| `notify` | 文件系统监听 |
| `image` | 图片处理、缩略图生成 |
| `kamadak-exif` | EXIF 解析 |
| `rawloader` | RAW 格式解码 |
| `lopdf` | PDF 解析 |
| `ttf-parser` | 字体解析 |
| `symphonia` | 音频解码 |
| `lofty` | 音频标签 (ID3/Vorbis) |
| `sha2` | 文件哈希 (SHA256) |
| `uuid` | ID 生成 |
| `chrono` | 时间处理 |
| `serde` / `serde_json` | 序列化 |
| `toml` | 配置文件解析 |
| `tracing` / `tracing-subscriber` | 日志/追踪 |
| `clap` | 命令行参数解析 |
| `thiserror` | 错误类型定义 |
| `anyhow` | 错误传播 |
