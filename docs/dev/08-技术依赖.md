# 08 - 技术依赖

## Rust Crates

| Crate | 用途 |
|-------|------|
| `axum` | HTTP 框架 |
| `tonic` | gRPC 框架 |
| `tonic-web` | gRPC-Web 支持 |
| `prost` | Protobuf 编解码 |
| `tokio` | 异步运行时 |
| `tower` | 中间件 |
| `tower-http` | CORS、日志、压缩等 HTTP 中间件 |
| `notify` | 文件系统监听 |
| `rusqlite` / `sqlx` | SQLite 访问 |
| `sea-query` | SQL 构建器 (数据库无关) |
| `image` | 图片处理、缩略图生成 |
| `webp` | WebP 编码 |
| `kamadak-exif` | EXIF 解析 |
| `rawloader` | RAW 格式解码 |
| `lopdf` | PDF 解析 |
| `ttf-parser` | 字体解析 |
| `symphonia` | 音频解码 |
| `lofty` | 音频标签 (ID3/Vorbis) |
| `sha2` | 文件哈希 (SHA256) |
| `uuid` | ID 生成 |
| `serde` / `serde_json` | 序列化 |
| `toml` | 配置文件解析 |
| `tracing` / `tracing-subscriber` | 日志/追踪 |
| `tauri` | 桌面应用框架 |
| `refinery` | 数据库迁移 |
| `thiserror` | 错误类型定义 |
| `anyhow` | 错误传播 |
| `tokio-tungstenite` | WebSocket |
