# 06 - 前端设计

## 技术栈

| 库 | 用途 |
|----|------|
| React 19 | UI 框架 |
| Vite 7 | 构建工具 |
| TypeScript 5 | 类型安全 |
| shadcn/ui + Tailwind CSS 4 | 组件库 + 样式 |
| zustand | 状态管理（含 API 数据缓存与加载状态） |
| react-router v7 | 路由 |
| i18next + react-i18next | 多语言 |
| fetch（`services/http.ts`） | REST API 请求 |
| WebSocket（`services/websocket.ts`） | 实时事件推送 |
| @tanstack/react-virtual | 虚拟列表（大量资源滚动） |

## 路由与页面（当前实现）

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | Home | 打开 / 创建资源库、最近资源库列表 |
| `/library` | Library | 当前资源库详情、关闭资源库 |
| `/system` | System | 服务在线状态、版本、API 地址、当前资源库路径 |
| `/settings` | Settings | 主题、语言 |

开发时 Vite（`:1420`）通过 proxy 转发 `/api`、`/health` 到后端；生产环境由 `ourboros server` / `ourboros app` 同端口托管 `web/dist`。

## 主题系统

- 基于 CSS 变量 + Tailwind CSS dark mode
- 支持：亮色 / 暗色 / 跟随系统
- shadcn/ui 原生支持主题切换
- 主题状态持久化到 localStorage

## 多语言 (i18n)

- i18next 管理翻译资源
- 初始支持：中文（zh）、英文（en）
- 翻译文件：`web/src/i18n/locales/{lang}.json`
- 语言切换持久化到 localStorage

## 数据请求策略

- **REST API**：通过 `services/http.ts` 封装 `fetch`，由 zustand store 直接调用并管理 `loading` / `error` 状态
- **实时事件**：通过 `services/websocket.ts` 连接 `/ws/events`，store 订阅事件后增量更新本地状态
- 不使用 `@tanstack/react-query`，避免引入额外的请求缓存层

## Zustand Store 设计

```typescript
// stores/library.ts - 资源库状态
interface LibraryStore {
  currentLibrary: LibraryInfo | null;
  recentLibraries: LibraryInfo[];
  open: (path: string) => Promise<void>;
  create: (path: string, name: string) => Promise<void>;
  close: () => Promise<void>;
  fetchInfo: () => Promise<void>;
}

// stores/system.ts - 服务状态
interface SystemStore {
  status: 'online' | 'offline' | 'unknown';
  systemInfo: SystemInfo | null;
  libraryDetail: LibraryInfo | null;
  checkedAt: string | null;
  refresh: () => Promise<void>;  // GET /health + /api/system/info + /api/library/info
}

// stores/assets.ts - 资源列表状态
interface AssetsStore {
  assets: Asset[];
  selectedIds: Set<string>;
  viewMode: 'grid' | 'list' | 'waterfall';
  sortBy: SortField;
  filters: FilterState;
  pagination: PaginationState;
  fetchAssets: () => Promise<void>;
  selectAsset: (id: string) => void;
}

// stores/sidebar.ts - 侧边栏状态
interface SidebarStore {
  folders: Folder[];
  smartFolders: SmartFolder[];
  tags: TagGroup[];
  activeItem: SidebarItem | null;
}

// stores/ui.ts - UI 状态
interface UIStore {
  theme: 'light' | 'dark' | 'system';
  locale: string;
  sidebarWidth: number;
  inspectorWidth: number;
  inspectorVisible: boolean;
}
```
