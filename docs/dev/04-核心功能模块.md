# 04 - 核心功能模块

## 文件监听 (File Watcher)

| 事件 | 处理 |
|------|------|
| 文件创建 | 自动索引，解析元数据，生成缩略图，写入数据库 |
| 文件删除 | 标记为已删除或从数据库移除，清理缩略图 |
| 文件移动/重命名 | 更新路径映射 |
| 文件修改 | 重新解析元数据，重新生成缩略图 |

**技术选型**：`notify` crate (跨平台文件系统事件监听)

**事件推送**：通过 WebSocket 实时推送到前端

**注意事项**：
- 使用 debounce 合并短时间内的重复事件
- 忽略 `.ourboros/` 目录自身的变化
- 忽略配置中指定的文件模式（如 .DS_Store）
- 大型资源库需注意 watcher 内存占用

## 元数据解析

| 资源类型 | 解析内容 | 技术选型 |
|----------|----------|----------|
| 图片 | EXIF (相机、光圈、快门、ISO、GPS)、尺寸、主色调 | `kamadak-exif` / `image` |
| PDF | 页数、标题、作者、创建日期 | `lopdf` |
| 字体 | 字体族名、样式、字重、字符集预览 | `ttf-parser` |
| 音频 | ID3 标签（标题、艺术家、专辑）、时长、BPM、采样率 | `symphonia` / `lofty` |
| 颜色 | 主色调提取、色板生成 | `image` + 自定义算法 |

## 缩略图生成

- 导入/检测到新文件时异步生成（后台线程池）
- 生成三种尺寸：200px / 400px / 800px（以短边为准）
- 统一存为 WebP 格式（体积小、质量好）
- 存放在 `.ourboros/thumbnails/` 目录
- 特殊处理：
  - PDF：取首页渲染为图片
  - 字体：渲染预览文本为图片
  - 音频：生成波形图
  - RAW：使用 `rawloader` 解码后生成

## 搜索与筛选

| 搜索维度 | 说明 |
|----------|------|
| 关键词 | 文件名、标签、备注全文搜索 (SQLite FTS5) |
| 标签 | 精确匹配/模糊匹配 |
| 颜色 | 按主色调/色系筛选 |
| 文件类型 | 按格式分类 |
| 尺寸 | 按图片分辨率范围 |
| 日期 | 按创建/修改/导入日期 |
| 星级评分 | 1-5 星 |
| 文件夹 | 按虚拟文件夹 |

## 资源组织

- **虚拟文件夹**：不移动文件，通过数据库关联实现。一个资源可属于多个虚拟文件夹。
- **智能文件夹**：基于规则自动聚合（如：所有 PNG + 星级>=3）
- **标签系统**：支持标签分组、批量打标签、自动标签
- **星级评分**：1-5 星
- **颜色标记**：快速分类
- **文件去重**：基于 SHA256 哈希检测重复文件
