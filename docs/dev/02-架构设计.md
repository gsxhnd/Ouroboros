# 02 - 架构设计

## 整体架构（系统托盘 + 内嵌 HTTP Server）

```
┌─────────────────────────────────────────────────────────┐
│                     Frontend (React)                      │
│     shadcn/ui + zustand + i18next + react-router        │
│         通过浏览器访问，HTTP REST 与 Backend 通信         │
└────────────────────────┬────────────────────────────────┘
                         │ HTTP REST
┌────────────────────────▼────────────────────────────────┐
│                   ourboros-server (Axum)                  │
│        REST API + WebSocket (文件监听事件推送)             │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│                    ourboros-core (Rust lib)               │
│  资源索引 │ 文件监听 │ EXIF解析 │ 缩略图生成 │ 搜索引擎   │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│              Storage Layer (SeaORM)                       │
│   SQLite (本地/默认) │ PostgreSQL │ MySQL (通过配置切换)   │
│   文件系统 (原始文件 + .ourboros/ 元数据目录)             │
└─────────────────────────────────────────────────────────┘
```

## 桌面端架构

```
┌──────────────────────────────────────────┐
│           ourboros-app (系统托盘)          │
│                                          │
│  tray-icon + muda (菜单) + tao (事件循环) │
│         │                                │
│         ├── 启动/停止 HTTP Server         │
│         ├── 打开浏览器访问界面            │
│         └── 退出应用                     │
│                                          │
│  内嵌 ourboros-server (随机端口)          │
└──────────────────────────────────────────┘
```

## 部署模式

| 模式 | 说明 |
|------|------|
| **桌面端 (系统托盘)** | `ourboros-app` 启动后驻留系统托盘，内嵌 Axum HTTP server（localhost:随机端口）。用户通过托盘菜单打开浏览器访问界面。macOS 打包为 .app bundle（LSUIElement），Windows 隐藏控制台窗口。 |
| **服务端** | 独立运行 `ourboros-server` 二进制，监听固定端口，Web 前端通过网络访问。适合 NAS / 团队共享场景。 |

## 平台适配

| 平台 | 适配方案 |
|------|----------|
| **macOS** | `.app` bundle + `Info.plist` 设置 `LSUIElement=true` + `ActivationPolicy::Accessory`，不显示 Dock 图标 |
| **Windows** | `#![windows_subsystem = "windows"]` 隐藏控制台窗口 |
| **Linux** | 通过 `.desktop` 文件启动，依赖 GTK3 显示托盘图标 |
