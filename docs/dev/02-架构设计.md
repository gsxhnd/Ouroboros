# 02 - 架构设计

## 整体架构（Tauri 内嵌 HTTP Server）

```
┌─────────────────────────────────────────────────────────┐
│                     Frontend (React)                      │
│     shadcn/ui + zustand + i18next + react-router        │
│         统一通过 HTTP/gRPC 与 Backend 通信                │
└────────────────────────┬────────────────────────────────┘
                         │ HTTP REST / gRPC-Web
┌────────────────────────▼────────────────────────────────┐
│                   ourboros-server (Axum)                  │
│        REST API + gRPC + WebSocket (文件监听事件推送)      │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│                    ourboros-core (Rust lib)               │
│  资源索引 │ 文件监听 │ EXIF解析 │ 缩略图生成 │ 搜索引擎   │
└────────────────────────┬────────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────────┐
│              Storage Layer                                │
│   SQLite (本地/默认) │ PostgreSQL (服务部署可选)           │
│   文件系统 (原始文件 + .ourboros/ 元数据目录)             │
└─────────────────────────────────────────────────────────┘
```

## 部署模式

| 模式 | 说明 |
|------|------|
| **桌面端 (Tauri)** | Tauri 进程启动时内嵌 Axum HTTP server（localhost:随机端口），前端 WebView 连接此 server。可多实例运行。 |
| **服务端** | 独立运行 `ourboros-server` 二进制，监听固定端口，Web 前端通过网络访问。适合 NAS / 团队共享场景。 |
